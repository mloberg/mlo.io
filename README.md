# mlo.io

Jekyll site for [mlo.io](http://mlo.io)

## Install

    bundle install --path vendor --standalone --deployment

## Build

    bundle exec jekyll build
    # Production mode
    JEKYLL_ENV=production bundle exec jekyll build

## Testing

    bundle exec htmlproof build
